## ðŸ“– How to Use

### 1. Create Notebooks
- On the home page, create notebooks to organize your subjects (e.g., "Math", "History").
- You can create sub-notebooks (e.g., Math > Algebra) by selecting the parent notebook before creating the new one.

### 2. Generate and Upload Questions
- Use the provided prompt models in the project root to generate questions using your favorite AI (ChatGPT, Gemini, Claude):
    - `question_model_en.md` (English)
    - `question_model_fr.md` (French)
    - `question_model_pt.md` (Portuguese)
- Copy the JSON generated by the AI.
- In the App, enter the desired notebook and paste the JSON into the upload area.

### 3. Study
- Click **Study** on the notebook card.
- Choose the mode:
    - **All**: All questions.
    - **Incorrect**: Only questions you previously got wrong.
    - **Unresolved**: Questions you haven't answered yet.
- Receive immediate feedback with the answer explanation.
